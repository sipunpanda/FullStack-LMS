# LMS FullStack

To Clone Project
```
git clone https://github.com/sipunpanda/FullStack-LMS
```


## LMS Frontend

### Setup instruction


1. Move into the directory
```
cd FullStack-LMS/lms-frontend
```
2. Install dependencies
```
npm install
```
3. Start the application
```
npm start
```

## LMS Frontend

### Setup instruction

1. Move into the directory
```
cd FullStack-LMS/lms-backend
```
2. Install dependencies
```

```
3. Start the application
